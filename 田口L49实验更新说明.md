# 田口L49实验系统更新说明

## 🔄 更新内容

### 主要变更
1. **实验重复次数**: 7次 → **10次**
2. **总实验量**: 343次 → **490次** (49组 × 10次)
3. **评价指标**: 增强5:3:2加权综合评价输出

---

## 📊 评价指标详情

### 5:3:2加权综合评价公式
```
综合得分 = 0.5 × 超体积(HV) + 0.3 × 反向世代距离(IGD) + 0.2 × 世代距离(GD)
```

### 指标归一化处理
- **超体积(HV)**: 直接归一化，假设最大值为1.21
- **IGD**: 转换为 `1/(1+IGD)`，越小越好→越大越好
- **GD**: 转换为 `1/(1+GD)`，越小越好→越大越好

### 信噪比计算
采用田口方法望大特性信噪比:
```
SNR = -10 × log10(mean(1/scores²))
```

## 📋 输出文件

### 结果目录结构
```
taguchi_results_YYYYMMDD_HHMMSS/
├── exp_01_run_01.json          # 单次实验结果
├── exp_01_run_02.json
├── ...
├── exp_01_summary.json         # 实验组汇总(10次平均)
├── exp_02_summary.json
├── ...
├── final_results.json          # 最终汇总结果
├── taguchi_analysis.json       # 田口分析结果
└── l49_results_summary.xlsx    # Excel格式汇总表
```

### Excel表格列说明
| 列名 | 说明 |
|------|------|
| `Exp_ID` | 实验组编号(1-49) |
| `A_LearningRate` | 学习率参数值 |
| `B_EpsilonDecay` | 探索率衰减参数值 |
| `C_GroupRatios` | 鹰群分组比例 |
| `D_Gamma` | 折扣因子参数值 |
| `HV_Mean/Std` | 超体积指标均值/标准差 |
| `IGD_Mean/Std` | 反向世代距离均值/标准差 |
| `GD_Mean/Std` | 世代距离均值/标准差 |
| `Comprehensive_Mean/Std` | **5:3:2加权综合得分均值/标准差** |
| `SNR_Value` | **信噪比(基于综合得分)** |
| `Runtime_Mean` | 平均运行时间 |
| `Successful_Runs` | 成功运行次数(最大10) |
